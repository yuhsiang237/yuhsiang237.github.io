API

應用程式介面（英語：Application Programming Interface），縮寫為API，是一種計算介面，它定義多個軟體中介之間的互動，以及可以進行的呼叫（call）或請求（request）的種類，如何進行呼叫或發出請求，應使用的資料格式，應遵循的慣例等。

簡單來說就是:程式的接口，可以傳送參數給後端處理的函式，之後會把所需的資料回傳你。
