SQL Server 常見SQL語法筆記

前言

在這裡整理一些專案常用的SQL語法。

語法

1.新增一筆資料

2.更新一筆資料

3.刪除一筆資料

4.查詢該表所有資料

5.使用查詢結果新增資料

6.INNER JOIN 主表跟次要表都有相同的值才撈出

7.LEFT JOIN 主表值為主嘗試撈取次要表值，次要表有值就撈，沒值就NULL

8.GROUP BY 取得以哪個欄位作為群，並且操作該群的集合資料

9.多對多資料處理(即使用left join/join的組合技)
